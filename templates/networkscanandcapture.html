Network Scan and Capture

Network Scan

Network Scan is the process of scanning the current network for open ports, protocols being run on them and vulnerabilities in the version of protocol being run. Network Scan is an effective measure to find vulnerabilities and resolve them before they lead to some serious attack

Network Capture/Sniff

Network Capture is the process of scanning incoming data packets and outgoing data packets for any kind of malicious activity. Sniffing the contents of data packets will help in detecting incoming virus or malware and sniffing outgoing data packets will help in detecting information and data leakage.
