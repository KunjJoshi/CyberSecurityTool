SQL Injection

SQL Injection is a code injection based attack, used to manipulate SQL Databases and get fake authentication. SQL Injection can be prevented using parametrization and it can be easily detected using code review. One can easily get access to either admin account or to user account using SQL Injection codes.

Reverse Shell

A Reverse Shell is an attack carried out by an attacker in a client computer, such that the host gets full command line access of the client. It is very easy to code and can be deployed using social engineering techniques. The Reverse Shell detection is equally easy and can be done through checking of logs and looking at whether or not malicious commands have been passed out of the network or not

Keylogger

A keylogger is a software which notes down each and every keystroke and the time of keystroke. It is very easy to implement, but very hard to detect. It is used for espionage and hijacking accounts of social media and other such accounts using hijacked passwords. Keylogger can be detected if a code segment is continuously using a file from outside the network.

Data Loss Prevention

Stopping data from getting into unauthorised hands is one of the most important and basic strategies of Cybersecurity. It is based on the pillar of cybersecurity called privacy. Most of the losses incurred by the companies due to security breaches are due to data leakage. Google Cloud Platform avails DLP tools and API which can be implemented to secure data from leakage.

SSL Certificate Details

A Secure Sockets Layer (SSL) Certificate is a Security Certificate issued by a known Certificate Authority and keeps a website safe and secure. SSL certificates are what enable websites to move from HTTP to HTTPS, which is more secure. An SSL certificate is a data file hosted in a website's origin server. SSL certificates make SSL/TLS encryption possible, and they contain the website's public key and the website's identity, along with related information.
